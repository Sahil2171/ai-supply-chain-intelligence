

# AI Supply Chain Intelligence: A Prescriptive Framework for Logistics and Inventory Optimization

[![Python](https://img.shields.io/badge/Python-3.14-blue.svg)](https://www.python.org/)
[![Machine Learning](https://img.shields.io/badge/Machine%20Learning-Prescriptive-orange.svg)]()
[![Status](https://img.shields.io/badge/Status-Active-success.svg)]()

Project Link: [Click Here](https://ai-supply-chain-intelligence.streamlit.app)

A data-driven web application built with Streamlit that leverages machine learning to optimize and analyze supply chain operations. 

**Project Timeline:** 15th January 2006 – 25 April 2026

## 🚀 Features
* **Demand Forecasting:** Predicts future product demand using historical data.
* **Delay Classification:** Identifies potential shipping delays before they happen.
* **Inventory Optimization:** Smart recommendations for inventory management.
* **Shipping Regression:** Estimates shipping times and costs.
* **Interactive Dashboards:** Visualizations powered by Plotly.

## 🛠️ Tech Stack
* **Frontend:** Streamlit
* **Data Processing:** Pandas, NumPy
* **Machine Learning:** Scikit-learn (Pickle for model serialization)
* **Visualization:** Plotly Express

## 💻 Local Installation & Setup

1. **Clone the repository:**
```bash
git clone [https://github.com/SAHIL2171/ai-supply-chain-intelligence.git](https://github.com/SAHIL2171/ai-supply-chain-intelligence.git)
cd ai-supply-chain-intelligence

```


2. **Create and activate a virtual environment:**
* **Windows:**
```bash
python -m venv .venv
.venv\Scripts\activate

```


* **Mac/Linux:**
```bash
python -m venv .venv
source .venv/bin/activate

```




3. **Install dependencies:**
```bash
pip install -r requirements.txt

```


4. **Run the application:**
```bash
streamlit run app.py

```
